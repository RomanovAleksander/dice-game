# Dice Game

A game built with Next.js 16, TypeScript, and Material-UI.

## Features

- Bet on dice rolls (OVER/UNDER a threshold)
- Interactive slider for setting bet thresholds
- Real-time game results
- Game history (last 10 games)

## Tech Stack

- **Next.js 16**
- **TypeScript**
- **Material-UI v7**
- **React 19**

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to play the game.

## Build for Production

```bash
npm run build
npm start
```

## Game Rules

1. Choose a threshold number (1-100)
2. Select OVER or UNDER
3. Click PLAY to roll the dice
4. Win if the result matches your bet!
